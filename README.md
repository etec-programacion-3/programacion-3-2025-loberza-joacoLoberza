## Como iniciar proyecto:
1. Ejecute esto en la raíz del proycto para instalar las dependencias:
    npm install
2. Luego, ejecute esto en la raíz del proyecto para iniciar el servidor backend:
    node src/backend/app.js
3. Luego, para abrir el frontend, puede iniciar el servidor de desarrollo Vite:
    cd src/frontend
    npm run dev
4. Finalmente, ingrese a su navegador con la siguiente url:
    http://localhost:5173
